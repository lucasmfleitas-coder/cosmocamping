Cosmocamping — Landing Page

Modern and responsive landing page developed for an astronomy camping and astrotourism project.

🌐 Live Demo

https://cosmocamping-rqnh.vercel.app/

Features

Responsive design (mobile-first)
Modern UI built with React
Component-based architecture
Smooth scrolling navigation
WhatsApp contact integration
Animated elements on scroll
Fixed dynamic navigation bar
Production deployment with Vercel

Installation and Setup:
Clone the repository:
git clone https://github.com/lucasmfleitas-coder/cosmocamping.git
Go into the project folder:
cd cosmocamping
Install dependencies:
npm install
Run development server:
npm run dev
Build for production
npm run build
